🆘 QUICK TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════════

  ❌ "npm not found"
     → Install Node.js from nodejs.org

  ❌ "Build failed"
     → Run: npm install
     → Then rebuild: npm run electron:build

  ❌ "Installer not created"
     → Check: release/ folder exists
     → Check disk space (need 5GB+)

  ❌ "Auto-update not working"
     → Make sure GitHub release is PUBLISHED (not draft)
     → Verify version in release matches package.json
